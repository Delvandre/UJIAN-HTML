Praktik Git: Clone, Branch, Commit, Push, dan Pull Request

Petunjuk Ujian:

  1. Lakukan Clone repository ini ke komputer lokal menggunakan Sourcetree.
  2. Pilih satu Slot Kotak sesuai nomor absen/kelompok (1-6).
  3. Ubah konten di dalam file index.html pada bagian komentar yang sesuai.
  4. Lakukan Commit dengan pesan yang jelas (Contoh: "Update Slot 1 - Nama Siswa").
  5. Push ke GitHub dan lakukan Pull Request ke branch utama.
